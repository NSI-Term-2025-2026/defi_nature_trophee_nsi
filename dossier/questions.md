# Questions 
 
## 1. Le projet est-il une création originale ?
Le projet est entièrement original. Cependant, nous avons utilisé des extraits de la documentation de Pygame pour certaines parties, notamment pour l’implémentation de fonctionnalités comme les sons ou le menu déroulant à gauche. Ces extraits ont été adaptés pour correspondre aux besoins du projet.

## 2. Avez-vous eu recours à l'IA ? Si oui, détaillez les modalités d’utilisation.
Oui, nous avons utilisé l’IA pour plusieurs aspects du projet :
Upscaling des images d'animaux : L’IA nous a aidés à améliorer la qualité de certaines images en les agrandissant sans perdre de détails.


Apprentissage de Pygame : Nous avons utilisé l’IA pour mieux comprendre certaines fonctionnalités avancées de Pygame, comme l’ajout de sons ou la création d’un menu déroulant.


Vérification des bonnes pratiques : L’IA nous a permis de vérifier que notre code respectait les bonnes pratiques de développement, comme l’utilisation des majuscules pour les classes et l’ajout des docstrings.


Contrôle des résultats statistiques : Nous avons aussi utilisé l’IA pour nous assurer que les résultats du module statistique étaient cohérants, notamment en comparant la médiane à la moyenne pour éliminer les valeurs aberrantes.


Apprentissage de GitHub : Nous avons utilisé l’IA pour nous familiariser avec l’utilisation de GitHub, comprendre les pull requests et créer un site GitHub Pages pour héberger le projet
